# Tableau Companion Dashboard

This folder is reserved for the Tableau version of the AI occupation adoption-gap dashboard.

The Tableau dashboard will use the same derived inputs in `input/derived_data/` and the same analytical views represented by the DuckDB SQL layer:

- Headline KPI summary
- Group-level exposure and adoption-gap comparison
- High-capability occupations with the largest adoption gaps
- Occupations where observed AI use is already highest
- Gap-band or quartile view for segmentation
